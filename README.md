# Snake Console Game

## Ein einfaches Snake-Spiel als Konsolenanwendung für Windows mit anpassbaren Spieleinstellungen und erweiterten Spielmechaniken.

## Features

- Klassisches Snake in der Windows-Konsole

- Konfigurierbares Spielfeld (Breite/Höhe)

- Anpassbare Snake-Farbe

- Einstellbare Geschwindigkeit

- Zwei Apfeltypen:

- Roter Apfel: erscheint regulär, verlängert die Snake um +3

- Lilaner Apfel: erscheint alle 30 Sekunden, verlängert um +9

- Highscore-System: speichert höchste erreichte Länge und überlebte Zeit lokal

- Live-Stats: aktuelle Länge & Highscore werden oben im Bildschirm angezeigt

## Kompatibilität

- Windows only

- Standalone .exe im latest release

- Optimiert für eine Konsolengröße von 80×25

- Größere Fenster funktionieren ohne Probleme

- Kleinere Fenster können Darstellungsprobleme verursachen
